# Mastermind: Pygame Edition

A classic code-breaking game built with Python and Pygame.

## How to Play
1. **Objective:** Guess the secret 4-color code generated by the computer.
2. **Controls:**
   - Click colors to fill your guess row.
   - Click **GUESS** to submit your attempt.
   - Click **DELETE** to remove the last peg.
3. **Feedback:**
   - **Black/Red Peg:** Right color in the right position.
   - **White Peg:** Right color in the wrong position.
